# 心理陪伴 · Mental Health Companion

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个温暖、非评判的 AI 心理陪伴 skill，专为中文用户和东亚文化背景设计。

A warm, non-judgmental mental health companion skill for Claude Code — designed for Chinese-speaking users with East Asian cultural sensitivity.

---

## 功能特点

- **温暖陪伴** —— 不是一个冷冰冰的评估工具，而像一个善于倾听的朋友
- **东亚文化敏感** —— 理解孝道、面子、内卷、催婚、讨好型人格等文化特有的压力来源
- **多用户档案** —— 按称呼自动创建/查找档案，一台电脑多人使用互不干扰
- **跨会话记忆** —— 自动记录关键心理状态，下次对话自然衔接，不用从头说起
- **通俗心理科普** —— 用比喻和日常语言解释心理机制（烟雾报警器、压力水杯、内心的裁判……），不堆砌学术术语
- **危机识别** —— 内置危机信号检测和中国大陆心理援助资源
- **隐私友好** —— 所有记录存在本地 skill 目录下，不上传任何地方，随时可删除

## 使用方式

在 Claude Code 中输入以下命令启动：

```
/mental-health-companion
```

第一次使用时，skill 会先问你的称呼，之后每次对话都会自动识别你、加载之前的记录。

你也可以直接表达情绪困扰，skill 会自动触发（但推荐用上面的命令，更稳定）：

- 「最近总是睡不好，脑子里一直想事情」
- 「我妈又打电话催我结婚了，好烦」

### 常用对话指令

| 你想做什么 | 这样说 |
|-----------|--------|
| 结束对话并保存 | 「今天就到这里」 |
| 清空自己的所有记录 | 「清空我的档案」 |
| 换个话题 | 直接说新的话题就好 |

## 重要声明

**本 skill 提供的是心理陪伴与情绪疏导，并非专业心理咨询或治疗。**

- ❌ 不是心理诊断或精神科诊断工具
- ❌ 不替代专业心理治疗或咨询
- ❌ 不是危机干预工具
- ✅ 是一个温暖的倾听者和陪伴者
- ✅ 提供情绪梳理和心理知识科普
- ✅ 指引专业求助的方向

**如果你正在经历严重的心理危机（自杀念头、自伤冲动等），请立即寻求专业帮助。** skill 内置了全国心理援助热线信息，也可以在 `references/危机资源.md` 中查看。

## 设计理念

### 为什么是「陪伴」而不是「诊断」

很多心理健康工具把用户当作「待评估的对象」，用问卷和量表去衡量「你有多严重」。这种方式在临床场景中有价值，但对一个只是想找人聊聊的人来说，它让人感觉被审视、被贴标签。

**这个 skill 选择了一条不同的路：**

- 先倾听，再回应 —— 不急于给建议
- 用比喻而不是术语 —— 让心理学变得可触摸
- 承认文化压力是真实的 —— 不只把问题归结为个人适应不良
- 记住你说过的话 —— 不用每次都从头讲起

### 为什么特别关注东亚文化

主流心理健康的语言和框架大多来自西方个体主义文化。在东亚文化中：

- 「设立边界」说起来简单，但面对父母时，「不孝」的道德压力是真实的
- 「做自己」是很好的建议，但当周围所有人都在卷的时候，停下来需要巨大的勇气
- 「抑郁症」这个词在很多家庭里仍然不能被提及

这个 skill 试图在**心理健康的专业框架**和**东亚文化的现实处境**之间找到一个平衡点。
